# 📊 Sales Dashboard

An interactive **Power BI Sales Dashboard** for analyzing sales performance through dynamic visualizations and KPIs.

## 🚀 Features
- Sales, Profit & Quantity analysis
- Regional and category-wise performance
- Monthly sales trends
- Interactive filters (Region, Category, Date)

## 📂 Files
- `sales_dashboard.pbit` – Power BI template

## 🛠️ Requirements
- Power BI Desktop
